# Telegram бот мониторинга Minecraft сервера

Бот для мониторинга сервера Minecraft 1.21.1 с автоматическими отчетами в Telegram.

## Функции

-  Регулярные отчеты каждый час (настраивается)
-  Мониторинг TPS сервера
-  Список игроков онлайн
-  Использование оперативной памяти
-  Загрузка процессора

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте бота в файле `monitoring.py`:
   - `TELEGRAM_BOT_TOKEN` - токен вашего бота от @BotFather
   - `CHAT_ID` - ID чата для отправки сообщений
   - `MINECRAFT_SERVER_IP` - IP адрес сервера Minecraft
   - `MINECRAFT_SERVER_PORT` - порт сервера (по умолчанию 25565)
   - `CHECK_INTERVAL` - интервал проверки в секундах (по умолчанию 3600 = 1 час)
   - `TPS_THRESHOLD` - порог TPS для уведомлений (по умолчанию 15)

## Как получить токен бота

1. Найдите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен

## Как узнать CHAT_ID

1. Отправьте любое сообщение вашему боту
2. Откройте в браузере: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
3. Найдите значение `"chat":{"id":...}`

## Запуск

```bash
python monitoring.py
```

## Команды бота

- `/start` - Информация о боте
- `/status` - Получить текущий статус сервера

## Примечание о TPS

Точное измерение TPS требует плагина на сервере. Бот использует оценку на основе задержки (latency).
Для точного TPS рекомендуется установить плагин типа Spark или аналогичный и интегрировать его API.
